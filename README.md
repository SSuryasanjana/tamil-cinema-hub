**Tamil Cinema Hub**

Tamil Cinema Hub is a modern and responsive movie showcase website dedicated to Tamil cinema. The application provides users with an engaging platform to explore popular Tamil movies, view movie details, browse by category, and enjoy an attractive user interface inspired by modern streaming platforms.

Features

* Responsive and mobile-friendly design
* Modern dark-themed user interface
* Tamil movie showcase with posters and descriptions
* Movie categorization (Action, Drama, etc.)
* Interactive movie cards with hover effects
* Movie details popup/modal functionality
* Smooth scrolling navigation
* Attractive hero section
* Easy deployment using GitHub Pages

 Technologies Used

* HTML5
* CSS3
* JavaScript
